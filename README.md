# asistente-comida-python
Asistente de Comida en Python

Este es mi primer proyecto de programación en Python con mis 17 años, proyecto individual, apenas aprendo a programar un poco.

 Descripción
El programa funciona como un asistente por consola que:
- Pregunta qué comida se antoja el usuario
- Pregunta el presupuesto
- Recomienda una opción según la lógica definida (Actualmente solo recomienda pizzas y hamburguesas de alto, medio y bajo presupuesto)

 Tecnologías
- Python 3
- Consola / Terminal

Cómo ejecutarlo
1. Clonar el repositorio
2. Ejecutar el archivo
